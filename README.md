# 化学实验台

一款面向高中化学学习的轻量游戏化实验平台。学生通过选择实验用品、组装仪器、拖动药品、观察实验现象、拼接并配平化学方程式，完成虚拟化学实验。

本项目重点关注不爱学习、基础较薄弱或对传统练习缺乏兴趣的学生，希望通过可视化实验和即时反馈降低化学学习门槛。

## 核心模块

### 有机化学

提供高中阶段常见有机化学反应实验，包括反应物选择、实验装置组合、实验现象观察和方程式配平。

### 无机化学

提供高中阶段常见无机反应实验，覆盖沉淀、气体、氧化还原、酸碱反应等内容。

### 综合实验

训练学生对实验用品、仪器用途、实验顺序、实验现象和化学方程式的综合判断能力。

### 快速闯关

每轮包含：

- 3道反应配对与现象题
- 2道方程式拼图题
- 1道分段综合实验题

学生全部一次答对后，可以解锁更高难度内容。

## 游戏流程

1. 选择实验。
2. 挑选需要的药品和仪器。
3. 在实验台上拖动并组装实验用品。
4. 按照正确顺序完成实验操作。
5. 观察容器内的实验现象。
6. 判断实验现象。
7. 选择反应物和生成物。
8. 配平化学方程式。
9. 查看实验评价和学习情况。

## 步骤反馈规则

每一个实验步骤都有独立的三次尝试机会，进入下一步骤后错误次数重新计算。

| 当前步骤错误次数 | 显示等级 | 完成提示 |
|---|---|---|
| 0次 | 完美完成 | 完美 |
| 1次 | 绿色 | 很好 |
| 2次 | 橙色 | 棒 |
| 3次 | 红色并公开答案 | 加油 |

如果一个实验中存在第三次尝试后才完成的步骤，该实验会被标记为“待重做”。

## 六类知识标签

系统计划根据学生在每个步骤中的错误内容，识别以下六类知识卡点：

- 反应物混淆
- 仪器用途混淆
- 现象混淆
- 方程式物质错误
- 配平错误
- 步骤顺序错误

这些标签将用于教师端的学习诊断，而不是简单记录学生的点击次数。

## 实验设计原则

- 实验用品和操作方式尽量符合高中化学教材。
- 气体通入液体时，先加入液体，再通过导气管缓慢通气。
- 气体燃烧实验使用储气瓶和导气管，不采用不合理的试管混合点燃方式。
- 不把铁钉未打磨、容器未清洗等基础准备工作设置为学生错误点。
- 透明溶液不虚构颜色变化。
- 必须借助指示剂观察的反应，会将加入指示剂作为实验步骤。
- 沉淀、气泡、液面、颜色和火焰等现象尽量在容器内部呈现。
