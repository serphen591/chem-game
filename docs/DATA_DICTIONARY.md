# 学习数据字典

## 记录边界

只记录学生被实际要求完成并点击“确认”的内容。不记录鼠标轨迹、无意义点击、铁钉打磨、容器清洗等由系统自动处理的基础准备动作。

## 六类知识标签

| 标签 | 识别来源 |
|---|---|
| 反应物混淆 | 固体、液体、气体药品错选、漏选或多选 |
| 仪器用途混淆 | 仪器、热源错选，或源/目标仪器用途不匹配 |
| 现象混淆 | 沉淀、气体、颜色、火焰等实验现象判断错误 |
| 方程式物质错误 | 反应物、生成物化学式所属一侧选择错误 |
| 配平错误 | 物质选择正确，但化学计量数错误 |
| 步骤顺序错误 | 用品正确但操作先后、通气、加热或组装顺序错误 |

同一次确认可以产生多个标签。例如同时错选烧杯和药品时，可同时标记“仪器用途混淆”和“反应物混淆”。

## 颜色等级

颜色由数据库根据当前步骤的 `step_error_count` 重新计算，客户端传入的颜色不作为可信数据：

- 第1次错误：`green`
- 第2次错误：`orange`
- 第3次错误：`red`，公开该步骤答案
- 进入下一步骤：错误次数重置
- 服务器最大只接受3，不会出现4/3

## 关键事件

| 事件 | 用途 |
|---|---|
| `attempt_started` | 建立一次实验尝试 |
| `step_error` | 保存错点、标签、期望值和实际选择 |
| `hint_opened` | 判断“提示后完成” |
| `answer_revealed` | 判断“答案后完成”和待重做 |
| `step_completed` | 保存步骤最终评级 |
| `attempt_completed` | 计算用时、完成方式和完成状态 |
| `attempt_abandoned` | 标记中途退出 |

## 教师端指标

- 完成率：有布置时按“已完成的学生-实验组合 ÷ 学生数×布置数”；无布置时暂按参与完成人数统计。
- 答案公开率：答案后完成次数 ÷ 已完成实验次数。
- 建议教师介入：同一标签跨3个实验出现，或存在红色公开答案记录。
- 待重做：任一步达到3次错误或公开答案。
